# MeloTTS-ONNX
MeloTTS-ONNX is the ONNX inference version of MeloTTS, specifically optimized for CPU real-time inference. The project supports:      ✅ Chinese-English mixed TTS     ✅ Multiple languages: Chinese, English, Japanese, Korean, Spanish, French, etc.     ✅ ONNX Runtime inference, fast inference speed
